Web App's first time use

By clicking on 'Web Application' button, you will redirected to the home page, if you are not logged in, you will be redirected to the login page. If it is your first time visiting the page, a new user will be automatically populated and then you will be able to login with the newly created account, with username and password of 'user', or register for a new account.

Once you login you will find yourself on Home Page and find that usage of this app is really easy, on top of each page there is navigation bar.

You can access your profile and update it by clicking on 'Profile'.

To create an event click on an empty time slot on 'Home' page day agenda or on any day on the calendar displayed on the 'Events' page. You will them be displayed with a pop up form to create an event. You can also click on existing event to check see more details about them, and if you are the owner of them you can also edit or delete them.

To search and view registered users click on 'Contacts' page. You will them see all registered users, you can also search for a specific user, you will notice that there is no search button, that's because as you type the search is conducted.You can also click on the Magnifying Glass Icon to see more details about a user.
